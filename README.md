# adebalogun.me

My personal website. I'm a software engineer based in New York. Before that I co-founded Hubbub, a rental marketplace for college students, and studied earth and environmental engineering at Columbia and chemical engineering at Imperial College London.

This site has my blog, project write-ups, and a few interactive things I've built — including a simulation of the old Hubbub website and a map of NYC's water supply system.

Built with React, TypeScript, and MUI. Hosted on GitHub Pages.
